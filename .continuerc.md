# FairArena - Continue.dev Context

## Project Overview

FairArena - Curl Tester provides ephemeral Docker sandboxes, a cURL/API
request builder and runner, and a real-time webhook inspector. It's designed
for secure demos and learning; sandboxes are resource-limited and ephemeral.

## Tech Stack

**Frontend:** React 19 + TypeScript, Vite, TailwindCSS

**Backend:** Node 22 + TypeScript, Express, node-pty, Docker

## Architecture

**API:** RESTful proxy endpoints (SSRF-protected), SSE for webhooks, WebSocket
for terminal sessions

## Code Standards

- Use TypeScript `strict` and prefer explicit types
- React components as functional hooks-based units
- Validate incoming API requests using `zod`

## Rules

1. TypeScript strict mode
2. Zod validation
3. No secrets in repo
